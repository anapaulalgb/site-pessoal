
# 🌐Site pessoal e Portfólio

Este é um projeto de site pessoal e portfólio desenvolvido como parte do curso da comunidade Dev Completo, com orientação da professora Bia.

O objetivo do projeto é criar uma presença online simples e profissional para apresentar minhas informações pessoais, habilidades e futuramente,projetos.

# 🛠️ Tecnologias utilizadas

•HTML5

•CSS3

Neste projeto não utilizamos frameworks ou bibliotecas externas, focamos apenas nos fundamentos da construção de páginas web.

# 📁 Estrutura do projeto
```bash
/index.html         -> Página principal do site
/styles.css          -> Arquivo de estilos CSS
/reset.css          -> Arquivo de reset dos estilos CSS
/assets/            -> Pasta para imagens e outros recursos
/sources/            -> Pasta para arquivos
```
# 📚 Aprendizados
Durante o desenvolvimento deste projeto, pude praticar:

Estruturação de páginas com HTML semântico

Estilização com CSS, incluindo uso de cores, fontes, layout com Flexbox, etc.

Organização de arquivos e boas práticas básicas de desenvolvimento web.
